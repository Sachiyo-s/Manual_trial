# PCA Hub 取引明細

PCA Hub 取引明細は、取引先に対して、請求書や納品書などの業務にかかわる書類を安全に配布することができるサービスです。<br/>
<br/>
※ PCA 商魂・商管シリーズ で説明していますが、 PCA hyper 会計シリーズ 債権管理オプション でもご利用いただけます。<br/>
PCA hyper 会計シリーズ 債権管理オプション では以下の点が異なります。<br/>
- 「納品書」には対応していません。<br/>
- マニュアル内の「得意先」は「請求先」と読み替えてください。<br/>
- 「請求先」は「先方担当者」には対応していません。<br/>

## 動作環境

詳細は [動作環境](../startup.md#動作環境) をご確認ください。<br/>
また、PCA Hub 取引明細はスマートフォンには対応しておりません。
