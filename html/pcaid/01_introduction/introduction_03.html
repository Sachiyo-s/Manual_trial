動作環境

  • 『PCA ID』 の各サービスは、以下の OS（オペレーティングシステム）と Web ブラウザーで動作可能です。

OS（オペレーティングシステム）

  • Microsoft Windows

  • Apple macOS

サポート対象

  • プラットフォームベンダーがサポートする OS バージョンをご利用ください。

  • 常に最新プログラムを適用し、OS ごとの最新バージョンをご利用ください。

Webブラウザー

  • Google Chrome

  • Microsoft Edge

  • Apple Safari

サポート対象

  • 常に最新プログラムを適用し、最新バージョンをご利用ください。

アプリケーションの動作機器

  • 『PCA ID』 の各アプリケーションは、以下の機器で動作確認し、快適に利用できるように努めています。

アプリケーション PC iPad Android タブレット スマートフォン
PCA ID 認証 ◯ ◯ ◯ ◯
PCA ID 管理コンソール ◯ ◯ ◯ ✕
PCA ID 組織作成 ◯ ◯ ◯ ✕
PCA ID アカウント設定 ◯ ◯ ◯ ◯

制限事項

  • PCA ID アプリケーションは、上記のうち比較的新しい世代の機器を対象としてサポートします。古くなった世代は、動作確認の対象から順次除外していきます。

  • スマートフォン対応はアプリケーションによって異なります。

  • すべての Android タブレットやスマートフォンでの動作を保証するものではありません。

  • iPad mini は動作確認しておらず、サポート対象外となります。
