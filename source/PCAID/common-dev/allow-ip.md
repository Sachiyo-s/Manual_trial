# 54. 許可IP（PoC・開発・検証環境）

## 前提

- 開発用のAWSアカウント（612816232952）における環境を対象とする
  - PoC環境（poc.id.pca-dev.net）
    - PCA Hubにおけるfree環境相当
    - PCAサブスクにおけるfeature環境相当
  - 開発環境（dev.id.pca-dev.net）
  - 検証環境（stg.id.pca-dev.net）
- CloudfrontのWAFで制御する

## IP制限ルール

1. pcaid-management-api (Lambda) から ALB の呼び出しなら素通りさせる
   - [CloudFront を経由するアクセス制御](./cloudfront-access.md)
   - 規定の Referer ヘッダー値が設定されている
2. BBSecのIPアドレスなら素通りさせる
   - ホワイトリスト以降のルールの対象外とする
3. それ以外のIPアドレスならホワイトリスト判定する
