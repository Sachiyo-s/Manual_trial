## 制限

- 明細の配信や管理は PCA Hub 取引明細のシステム管理者のみが行うことができます。<br/>
- 明細が公開されると取引先に公開メールが配信されます。
サーバーの状況により、指定された日時よりも遅れてメールが配信される場合があります。<br/>

## 明細をアップロードする

印刷指示の出力先の選択で「PCA Hub 取引明細」を選択して実行すると、「PCA Hub 取引明細」へアップロードすることができます。<br/>
  - [請求書、支払明細書をアップロードする](#請求書支払明細書をアップロードする)
  - [納品書をアップロードする](#納品書をアップロードする)
  - [アップロードできるファイルに関する制限](#アップロードできるファイルに関する制限)
  - [公開日時に関する注意事項](#公開日時に関する注意事項)

### 請求書、支払明細書をアップロードする
1. 「請求書」をアップロードする場合は、PCA 商魂・商管シリーズの「販売管理」-「請求」-「請求書」を起動します。<br/>
「支払明細書」をアップロードする場合は、「仕入管理」-「仕入締」-「支払明細書」を起動します、操作の流れは同様です。<br/>

2. 条件を設定し、実行します。<br/>
請求書、支払明細書をアップロードするには、印刷等を行う必要があります。<br/>
請求形態を、以下のどちらかの組み合わせに設定します。<br/>
    - 「本発行」「再発行」「仮発行」を選択する<br/>
    - 「請求締切」「随時請求」を選択し、「印刷等も行う」にチェックを入れる<br/>
    
    上記以外の組み合わせでは、アップロードできません。<br/>
    請求形態や条件の詳細については、PCA 商魂・商管シリーズのマニュアルをご確認ください。<br/>

   ![1](../assets/system_admin2.image1.png)<br/>

3. 「出力先の選択」で「PCA Hub 取引明細」を選択して [実行] ボタンをクリックします。<br/>
「ファイル名に付加する項目」にてアップロードされるファイル名に任意の情報を付けることができます。<br/>
![2](../assets/system_admin2.image2.png)<br/>

4. 明細の配信設定を行います。<br/>
   1. 「メール送信文」で登録した、メール送信文を設定します。<br/>
   2. PCA Hub 取引明細上で明細を公開およびメール配信したい日時を設定します。<br/>
   公開日時は、PCA Hub 取引明細の契約期間内で設定してください。<br/>
   詳細は、PCA 商魂・商管シリーズのマニュアルをご確認ください。<br/>
   ![3](../assets/system_admin2.image3.png)<br/>
   ### ポイント<!-- omit in toc -->
   - 「配信データの確認画面を表示する」にチェックを入れると、アップロードする内容を確認できます。<br/>
   PDF配信の場合はチェックを外すことで、確認を省略できます。<br/>
   アップロード内容を確認し、[実行]ボタンをクリックします。<br/>
   ![12](../assets/system_admin2.image12.png)<br/>

5. アップロード処理が正常に完了すると配信結果ダイアログが閉じます。<br/>
![4](../assets/system_admin2.image4.png)<br/>

### 納品書をアップロードする
「納品書」は一括アップロードする処理と、一枚ずつアップロードする処理があります。<br/>

- 一括でアップロードする場合は、PCA 商魂・商管シリーズの「販売管理」-「売上」-「納品書の一括発行」を起動します。<br/>
    1. 印刷等を実行します。<br/>
詳細は、PCA 商魂・商管シリーズのマニュアルをご確認ください。<br/>
    2. 以降の操作は [請求書、支払明細書をアップロードする](#請求書支払明細書をアップロードする) の **3** 以降 と同様です。<br/>
- 一枚ずつアップロードする場合は、PCA 商魂・商管シリーズの「販売管理」-「売上」-「売上伝票入力」を起動します。<br/>
  1. 「ファイル」-「納品書の発行方法」をクリックし、「その他の設定」-「印刷前に印刷指示を表示」にチェックを入れて設定してください。<br/>
印刷前に「印刷指示」を表示することで、納品書の出力先に「PCA Hub 取引明細」を選択することができます。<br/>
詳細は、PCA 商魂・商管シリーズのマニュアルをご確認ください。<br/>
![13](../assets/system_admin2.image13.png)<br/>

  2. 納品書の発行を行います。<br/>
以下のどちらかの方法で発行します。<br/>
詳細は、PCA 商魂・商管シリーズのマニュアルをご確認ください。<br/>
     - 売上伝票を表示し、ツールバーの「納品書」をクリックする。
     - 「納品書の発行方法」で「入力都度の発行」にチェックを入れて設定し、売上伝票登録時に自動発行させる。

  3. 「出力先の選択」で「PCA Hub 取引明細」を選択して [実行] ボタンをクリックします。<br/>
  「配信先の分類」にて納品書を送る配信先を指定できます。<br/>
  「ファイル名に付加する項目」にてアップロードされるファイル名に任意の情報を付けることができます。<br/>
  詳細は、PCA 商魂・商管シリーズのマニュアルをご確認ください。<br/>
  ![14](../assets/system_admin2.image14.png)<br/>

  4. 以降の操作は [請求書、支払明細書をアップロードする](#請求書支払明細書をアップロードする) の **4** 以降 と同様です。<br/>

### アップロードできるファイルに関する制限

|項目名|上限値|制限|備考|
|--|--|--|--|
|ファイル名|250文字|下記の記号を禁止<br/>・円マーク（&yen;）<br/>・スラッシュ（/）<br/>・コロン（:）<br/>・アスタリスク（*）<br/>・クエスチョンマーク（?）<br/>・ダブルクォート（"）<br/>・小なり（<）<br/>・大なり（>）<br/>・パイプライン（&#124;）|大文字と小文字や全角と半角を区別しません。|
|サイズ|15MB|||

### ポイント<!-- omit in toc -->

- 明細の配信設定は、一度にアップロードされるすべての明細に対して同一の設定が反映されます。<br/>
- 「メール送信文」は 、「前準備」-「メール送信文の登録」から登録します。<br/>

### 公開日時に関する注意事項

- 明細をアップロードする際は、公開日時をPCA Hub 取引明細の契約期間内に設定してください。<br/>
  - PCA Hub 取引明細の契約期間内のみ、クライアントサイトおよび管理サイトを閲覧することができます。<br/>
契約終了後は、明細の閲覧およびダウンロード等、全ての操作が行えません。<br/>
  - 公開日時に契約終了後の日時を指定した場合でも、メールおよびデジタルインボイスが送信される可能性があります。<br/>
契約終了後であるため、配信メールを受け取った配信先ユーザーは明細をダウンロードすることができません。<br/>
- サーバーの状況により、指定された日時よりも遅れてメールおよびデジタルインボイスが配信される場合があります。<br/>
- サーバーの状況により、メールの配信される日時とデジタルインボイスがダウンロード可能になる日時が前後する場合があります。<br/>

## 明細の配信状況を確認する

1. PCA 商魂・商管シリーズの「随時」-「配信状況の確認」を起動します。<br/>
2. 確認したい状況に合わせて「未公開」「公開済み」「リマインド」タブを開いてください。<br/>
![5](../assets/system_admin2.image5.png)<br/>

## 明細のダウンロード状況を確認する

明細を受け取った配信先ユーザーが、明細をダウンロードしたかどうかを確認することができます。<br/>
<br/>

1. PCA Hub 取引明細の管理サイトにアクセスします。<br/>
2. 「配信明細一覧」ページにアップロード済みの明細一覧が表示されます。<br/>
3. 「DL状況」を選択し、ダウンロード状況を確認します。<br/>
![6](../assets/system_admin2.image6.png)<br/>

### ポイント<!-- omit in toc -->
- 「DL状況」列が表示されていない場合は、列設定を変更します。<br/>
  1. 列名の下のドロップダウンから、「DL状況」をクリックします。<br/>
  ![7](../assets/system_admin2.image7.png)<br/>
- PCA 商魂・商管シリーズの「随時」-「配信状況の確認」の、「公開済み」タブの「ダウンロード日時」列からも確認することができます。<br/>
![8](../assets/system_admin2.image8.png)<br/>

## 明細の公開日時を変更する

未公開の明細は、アップロード後に公開日時を変更することができます。<br/>

1. PCA 商魂・商管シリーズの「随時」-「配信状況の確認」を起動します。<br/>
2. 「未公開」タブを開き、変更したい明細にチェックを入れます。<br/>
  ![9](../assets/system_admin2.image9.png)<br/>
3. 「公開日時」ボタンを押し、公開日時を変更します。<br/>
  ![10](../assets/system_admin2.image10.png)<br/>

## アップロードした明細を削除する

1. PCA 商魂・商管シリーズの「随時」-「配信状況の確認」を起動します。<br/>
2. 削除したい明細のあるタブを開きます。<br/>
3. 削除したい明細にチェックを入れ、「削除」メニューを実行します。<br/>
![11](../assets/system_admin2.image11.png)<br/>

### ポイント<!-- omit in toc -->
- 配信データを削除しても、「Peppol ネットワーク」に送信されたデジタルインボイスデータはキャンセルまたは削除されません。<br/>
