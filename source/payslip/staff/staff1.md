# 給与明細を受信する準備を行う<!-- omit in toc -->

PCA Hub 給与明細で明細を受信するための準備を行います。

## 目次<!-- omit in toc -->

- [制限](#制限)
- [PCA Hub アカウントの初期設定を行う](#pca-hub-アカウントの初期設定を行う)
- [同意内容を確認する](#同意内容を確認する)

## 制限

- PCA Hub のアカウントが作成されており、PCA Hub に招待されている必要があります。
- PCA Hub 給与明細のライセンスを付与されている必要があります。

## PCA Hub アカウントの初期設定を行う
### はじめて PCA Hub アカウントを登録するときは

 PCA Hub アカウントの初期設定を行います。

`【PCAサービス】≪会社名≫ からのアカウント設定リクエスト` というタイトルのメールを開いてください。<br/>
`≪会社名≫` には、PCA Hub 給与明細を契約している会社名が入ります。<br/>

[PCA Hub にログインするための手順 ＞ [PCA ID]をご利用の場合の操作 ＞ 1. 招待メールの受信](https://pca.jp/area_support/manual/hub/04_client/client_01.html?tocpath=%E3%82%AF%E3%83%A9%E3%82%A4%E3%82%A2%E3%83%B3%E3%83%88%E3%81%AE%E6%93%8D%E4%BD%9C%7C_____2#PCAID%E3%82%92%E3%81%94%E5%88%A9%E7%94%A8%E3%81%AE%E5%A0%B4%E5%90%88%E3%81%AE%E6%93%8D%E4%BD%9C) を確認し、手続きを行ってください。<br/>

### `【PCAサービス】{会社名} からのアカウント設定リクエスト` というメールが見つからないときは

- `【PCA Hub サービス】≪会社名≫からのパスワード設定リクエスト` が届いている場合<br/>
[PCA Hub にログインするための手順 ＞ [PCA ID]をご利用でない場合の操作 ＞ 1. 招待メールの受信](https://pca.jp/area_support/manual/hub/04_client/client_01.html?tocpath=%E3%82%AF%E3%83%A9%E3%82%A4%E3%82%A2%E3%83%B3%E3%83%88%E3%81%AE%E6%93%8D%E4%BD%9C%7C_____2#PCAID%E3%82%92%E3%81%94%E5%88%A9%E7%94%A8%E3%81%A7%E3%81%AA%E3%81%84%E5%A0%B4%E5%90%88%E3%81%AE%E6%93%8D%E4%BD%9C) を確認し、手続きを行ってください。<br/>

- `【PCAサービス】メールアドレス確認リクエスト` が届いている場合<br/>
[一般ユーザー向け ＞ メールでメールアドレス確認リクエストが届いたら](https://pca.jp/area_support/manual/pcaid/03_client/client_08.html) を確認し、手続きを行ってください。<br/>

#### ポイント<!-- omit in toc -->
- 対応しようとしたら、期限が切れていた場合<br/>
招待から一定時間経過していると、招待期限が切れてしまいます。<br/>
管理者へ再送信を依頼してください。<br/>
- 設定したパスワードを忘れてしまった場合<br/>
ログイン画面に、「パスワードを忘れた場合はこちら」や「パスワードをお忘れの方」というリンクがあります。<br/>
こちらから、ご自身でパスワードの再設定ができます。<br/>
参考）[パスワードの作り方](https://pca.jp/area_support/manual/pcaid/01_introduction/introduction_07.html)
- `【PCA Hub 給与明細】≪会社名≫ からのライセンス付与のお知らせ` というメールは何か<br/>
PCA Hub 給与明細を利用するためのライセンス付与のお知らせです。<br/>
この後の [同意内容を確認する](#同意内容を確認する) で使用します。<br/>

## 同意内容を確認する

給与担当者から PCA Hub 給与明細で明細を配信することに対する同意案内が配信されています。<br/>
同意事項に同意しないと PCA Hub 給与明細をご利用いただけません。

1. PCA Hub 給与明細のクライアントサイトにアクセスします。<br/>
  - PCA Hub からのライセンス付与メールに記載されているURLからアクセスする場合、URLをクリックします。<br/>
  ![1](../assets/staff1.image1.png)<br/>
  - ポータルサイトからアクセスする場合、[PCA Hub 給与明細] ボタンをクリックします。<br/>
  ![3](../assets/staff1.image3.png)<br/>

2. 同意案内および同意事項の全文を閲覧した上で「同意事項を確認しました。」のチェックをONにします。 <br/>
「同意事項」はスクロールして全文を表示する必要があります。<br/>
![2](../assets/staff1.image2.png)<br/>

3. [同意する] ボタンをクリックします。

### ポイント<!-- omit in toc -->
- `【PCA Hub 給与明細】≪会社名≫ からのライセンス付与のお知らせ` というタイトルのメールが見つからない、誤って削除したかもしれない場合<br/>
テナント名がわかる場合は、[URL](https://pca.jp/area_support/manual/hub/01_introduction/introduction_4.html?tocpath=%E3%81%94%E5%88%A9%E7%94%A8%E3%81%AE%E5%89%8D%E3%81%AB%7C_____4) を参考に給与明細クライアントサイトにアクセスすることができます。<br/>
テナント名がわからない場合は、管理者にお問い合わせください。<br/>
