# 組織で利用するサービスを取り消す

:::info 権限による制限

利用サービスの取り消しは、組織管理者のみ行うことができます。

:::

## (1) 管理コンソールへのアクセス

PCA ID 管理コンソールにアクセスするため、ブラウザーで、<https://id.pca.jp/orgs> にアクセスします。  
管理コンソールの画面を表示するには、組織管理者の権限が必要となります。

## (2) 組織の選択

現在の組織を確認し、必要があれば組織を選択します。

組織管理者の権限をもつ複数の組織に所属している場合、処理を実行したい組織を選択します。  
アクセス直後は、前回の組織が選択されています。

![組織の選択](../images/組織の選択.png)

## (3) カテゴリの選択

［組織管理］カテゴリを選択します。

![組織管理カテゴリ](./images/組織管理カテゴリ.png)

## (4) 利用サービスの取り消し

現在の組織において、利用サービスを取り消します。  
取り消したい利用サービスの右端にある、取り消しアイコン ![取り消しアイコン](./images/取り消しアイコン.png) をクリックします。

現在の組織から利用サービスを取り消すと、対象サービスと連携できない状態になります。  
組織情報の利用可能サービス一覧には、表示されなくなります。

![利用サービスの取り消し](./images/利用サービスの取り消し.png)

:::info 利用サービスの扱い

利用サービスは、追加と取り消しを何度でも繰り返すことができます。  
また、ある組織から利用サービスを取り消して、別の組織へ追加することも可能です。  
ただし、契約IDが同じ利用サービスを、複数の組織に追加することはできません。

:::
