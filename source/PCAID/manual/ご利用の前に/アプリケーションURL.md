# アプリケーション URL

『PCA ID』 アプリケーションの URL、および説明は以下のとおりです。

| アプリケーション | URL | 備考 |
| --- | --- | --- |
| PCA ID 管理コンソール | <https://id.pca.jp/orgs> | ・ PCA ID を利用するサービス経由であれば、サービスがログイン先の組織を指定します。<br/>・ 直接アクセスであれば、ユーザー自身がログイン先の組織を指定します。 |
| PCA ID 組織作成 | <https://id.pca.jp/orgs-new> | ・ PCAクラウド／PCAサブスクからアクセス可能です。 |
| PCA ID アカウント設定 | <https://id.pca.jp/account> | ・ PCA ID を利用するサービス経由であれば、サービスがログイン先の組織を指定します。<br/>・ 直接アクセスであれば、ユーザー自身がログイン先の組織を指定します。 |

:::tip ポイント

PCA ID アプリケーション、および PCA ID を利用するサービスに関する操作は、それぞれのマニュアルをご覧ください。

:::

:::info ログイン先の組織指定

ユーザー自身がログイン先の組織を指定する方法については、「[組織付きログイン名の入力](../ログインの流れ/組織付きログイン名の入力.md)」をご確認ください。

:::
