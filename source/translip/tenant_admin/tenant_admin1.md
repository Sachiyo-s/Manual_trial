# 実務担当者のアカウントを作成する<!-- omit in toc -->

PCA Hub 取引明細を利用するには、PCA Hub のアカウントが必要です。<br/>
PCA Hub のアカウントは個別または一括で作成することができます。<br/>
PCA Hub アカウントが作成されると、アカウントの初期設定を行うためのメールが配信されます。<br/>

## 制限

- PCA Hub のアカウントの作成はテナント管理者のみが行うことができます。<br/>

## 手動でアカウントを作成する

1. テナント管理サイトのユーザー一覧にアクセスします。<br/>
2. [新規] ボタンをクリックして新規入力ダイアログを表示します。<br/>
3. 基本情報の必須項目を入力します。<br/>
4. [保存] ボタンをクリックして入力内容を保存します。<br/>

![1](../assets/tenant_admin1.image1.png)<br/>

## CSV ファイルからアカウントを作成する

1. [PCA Hub ユーザー CSV ファイルフォーマット](https://pca.jp/area_support/manual/hubedoc/03_manager/manager_12.html) で、CSVファイルを作成します。
2. テナント管理サイトのユーザー一覧にアクセスします。
3. アクションメニューから「ユーザーのインポート」をクリックして CSV ファイルを選択します。<br/>
![2](../assets/tenant_admin1.image2.png)<br/>

4. インポートが完了して結果ログがダウンロードされるまでしばらく待ちます。
5. ダウンロードされたログファイルを開いてインポート結果を確認します。<br/>
![3](../assets/tenant_admin1.image3.png)<br/>

## PCA Hub アカウントが作成されたら

各実務担当者は、配信されたメールから初期設定を実行してください。<br/>
初期設定手順は [PCA Hub にログインするための手順](../../startup.md#pca-hub-にログインするための手順) をご覧ください。
