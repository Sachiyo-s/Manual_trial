# テナント管理者を追加する<!-- omit in toc -->

ご契約テナントのテナント管理者を設定します。<br/>
テナント管理者は必ず1名以上を設定してください。<br/>
テナント管理者はユーザーの管理やライセンス付与等のPCA Hub全般の設定・管理を行います。設定は慎重に行ってください<br/>

## 目次<!-- omit in toc -->

- [制限](#制限)
- [テナント管理者を追加する](#テナント管理者を追加する)

## 制限

- テナント管理者の追加は、テナント管理者のみが行うことができます。

## テナント管理者を追加する

1. テナント管理サイトのサービス一覧にアクセスします。<br/>
2. 「テナント管理者」を選択します。<br/>
3. [編集] ボタンをクリックして任意のアカウントを追加します。<br/>
4. [保存] ボタンをクリックして設定内容を保存します。<br/>
