PCA ID 組織を作成する

各種サービスから PCA ID を利用するには、まず最初に、お客様ごとの「PCA ID 組織」を作成する必要があります。
組織について詳しくは、「PCA ID 組織とは？」をご確認ください。

権限による制限

PCA ID 組織の作成は、PCA 製品のシステム管理者のみ行うことができます。

PCA Hub での組織

PCA Hub においては、PCA Hub テナントの作成と同時に、自動的に PCA ID 組織が作成されます。

PCA ID 組織の作成は、PCA 製品のシステムツール『連携アカウントでログオンの有効化』処理からおこないます。
「PCA ID 連携コードの登録」画面で［PCA ID 組織作成］ボタンをクリックし、『PCA ID 組織作成』の Web ページへ移動します。
すでに PCA ID 組織作成が完了し、PCA 製品と連携する状況であれば、［連携コード取得］を行ってください。

既存組織がある場合

PCA Hub を導入済みで既存組織がある場合、新たに PCA 製品を利用サービスとして追加するには、［PCA ID サービス登録］から行ってください。

PCA ID 組織作成

PCA ID 組織作成を開始すると、ブラウザーが起動して、次の画面が表示されます。

画面の指示に従い、契約確認のためにサービスの認証をおこないます。
その後、組織情報と組織管理者を入力して組織を作成します。

組織を作成すると、組織管理者として入力したユーザーのメールアドレスへリクエストメールが送信されます。
リクエストメールの指示に従って操作が完了すると、作成した組織へログインできるようになります。

作成したユーザーに対して、 【PCA サービス】≪組織表示名≫ からの2種類のリクエストメールのうち、状況に応じて、以下のいずれかが届けられます。

作成したユーザーが、新規アカウントであれば、アカウント設定リクエストを送信します。

リクエストのやり直し

アカウント設定リクエストの有効期限は1週間です。
リクエストの期限切れやメールアドレスの宛先違い等により、メールを再送信したい場合には、もう一度最初から組織作成の操作が必要となります。

すでに別の組織に所属している既存アカウントであれば、メールアドレス確認リクエストを送信します。
