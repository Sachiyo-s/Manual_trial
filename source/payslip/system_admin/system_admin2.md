## 制限

- 明細の配信や管理は PCA Hub 給与明細のシステム管理者のみが行うことができます。
- 明細が公開されると社員に公開メールが配信されます。<br/>
サーバーの状況により、指定された日時よりも遅れてメールが配信される場合があります。<br/>

## 明細をアップロードする

1. PCA給与シリーズの「給与」-「給与明細書」を起動します。
2. 「出力の種類」で「印刷等」を選択し、配信を行いたい社員を指定して [印刷指示] ボタンをクリックします。<br/>
![1](../assets/system_admin2.image1.png)<br/>

3. 「出力先の選択」で「明細配信」を選択して [実行] ボタンをクリックします。<br/>
「PDFファイル名の設定」にてアップロードされるファイル名を変更することができます。<br/>
![2](../assets/system_admin2.image2.png)<br/>

4. 明細の配信設定を行います。
   1. PCA Hub 給与明細上で明細を公開およびメール配信したい日時を設定します。
   2. 配信時に送信されるメールの件名と本文を設定します。
   3. 明細以外に同時に配信したいファイルがあれば「添付ファイル」を社員ごとまたは一括で指定します。<br/>
   ![3](../assets/system_admin2.image3.png)<br/>
   ![4](../assets/system_admin2.image4.png)<br/>

5. アップロード処理が正常に完了したことを確認します。<br/>
![5](../assets/system_admin2.image5.png)

### アップロードできる明細の種類

- 給与明細書
- 賞与明細書
- 源泉徴収票
- 給与改定通知書
- 標準報酬決定通知書
- 標準報酬改定通知書
- 年末調整通知書
- 還付金明細書
- [個人住民税特別徴収税額通知書（納税義務者用）](system_admin4.md)
- その他（任意のファイル）

### ポイント<!-- omit in toc -->

- 明細をアップロードして配信するためには、配信対象の社員が明細配布の同意案内に対して事前に同意している必要があります。
- 給与明細などの明細とは別に、任意のファイルを同梱して同時に配信することが可能です。
- 明細の配信設定は、一度にアップロードされるすべての明細に対して同一の設定が反映されます。
- PCA Hub 給与明細にアップロードされる実ファイルの名前（**ファイル名**）と、PCA Hub 給与明細の画面上で確認できるファイルの名前（**書類名**）は異なります。

### 公開日時に関する注意事項

- 明細をアップロードする際は、公開日時をPCA Hub 給与明細の契約期間内に設定してください。<br/>
  - PCA Hub 給与明細の契約期間内のみ、クライアントサイトおよび管理サイトを閲覧することができます。<br/>
契約終了後は、明細の閲覧およびダウンロード等、全ての操作が行えません。<br/>
- サーバーの状況により、指定された日時よりも遅れてメールが配信される場合があります。<br/>

## 明細の配信状況を確認する

1. PCA給与シリーズの「給与」-「給与明細書」を起動します。
2. 「配信状況」メニューを実行します。<br/>
![6](../assets/system_admin2.image6.png)<br/>

3. 年間の配信状況を確認したい場合には「年間表示」メニューを実行します。<br/>
![7](../assets/system_admin2.image7.png)<br/>

## 明細の閲覧状況を確認する

明細を受け取った社員が、明細を閲覧したかどうかを確認することができます。<br/>
<br/>

1. PCA Hub 給与明細の管理サイトにアクセスします。
2. 「社員一覧」ページにて閲覧状況を確認したい任意の社員をクリックします。
3. アップロード済みの明細一覧が表示されます。<br/>
既読の場合は細字で、未読の場合は太字で表示されます。<br/>
![8](../assets/system_admin2.image8.png)

## 明細の配信設定を変更する

配信済みのデータを直接編集することはできません。<br/>
[明細をアップロードする](#明細をアップロードする) の手順で同じ明細を再度アップロードすることにより、既存の配信設定を上書きすることができます。

## 配信した明細を削除する

1. PCA給与シリーズの「給与」-「給与明細書」を起動します。
2. 「配信状況」メニューを実行します。
3. 明細を削除したい社員を選択して「削除」メニューを実行します。<br/>
![9](../assets/system_admin2.image9.png)<br/>
