社員情報を確認する

PCA Hub 労務管理に登録されている社員情報を確認できます。
編集・削除も可能です。

制限

  • PCA Hub 労務管理のシステム管理者のみが行うことができます。

  • 「社員マスター システム管理者」のグループのメンバーである必要があります。

社員情報を確認する

PCA Hub 労務管理に登録されている社員情報を確認できます。

(1)  PCA Hub 労務管理の管理サイトにアクセスします。

(2)  「社員一覧」ページにて、確認したい社員をクリックします。

(3)  選択した社員情報の詳細が表示されます。

ポイント

  • 「社員一覧」ページでは、画面右上の検索欄で絞り込みを行うことができます。
    「社員番号」「氏名」「氏名フリガナ」に対して検索が行われます。

  • 「本人情報」以外の項目は、それぞれタブを選択することで内容が表示されます。

社員情報を編集する

PCA Hub 労務管理に登録されている社員情報を編集できます。

(1)  PCA Hub 労務管理の管理サイトにアクセスします。

(2)  「社員一覧」ページにて、確認したい社員をクリックします。

(3)  選択した社員情報の詳細画面にて、「編集」をクリックします。

(4)  入力内容の編集を行って、「保存」をクリックすると社員情報が更新されます。

ポイント

  • 「本人情報」や「住所」などそれぞれに「PCA給与連動用管理項目」があります。
    PCA給与で社員情報を受け入れる際に利用される項目のため、社員情報編集時にはこれらの確認・入力も行う必要があります。
    詳しくはPCA給与で社員情報を受け入れるをご覧ください。

社員情報を削除する

PCA Hub 労務管理に登録されている社員情報を削除できます。

(1)  PCA Hub 労務管理の管理サイトにアクセスします。

(2)  「社員一覧」ページにて、削除したい社員をクリックします。

(3)  「この社員を削除」をクリックすると、PCA Hub 労務管理から社員情報が削除されます。

ポイント

  • PCA Hub 労務管理から完全に削除されるため、元に戻すことはできません。

  • 社員情報が削除された社員は、クライアントサイトを利用できなくなります。

  • 社員情報を登録する場合は PCA給与から社員情報をアップロードする を参考にPCA Hub 労務管理に社員情報を登録してください。
