# 社員のアカウントを作成する<!-- omit in toc -->

PCA Hub 給与明細を利用するには、PCA Hub のアカウントが必要です。<br/>
PCA Hub アカウントは、テナント管理者が作成することもできますが、社員が自身で作成することもできます。<br/>
PCA Hub アカウント作成が完了すると、。アカウントの初期設定を行うためのメールが配信されます<br/>

## 制限

- PCA Hub のアカウントの作成はテナント管理者のみが行うことができます。

## 社員のアカウントを作成する

複数の社員アカウント作成方法があります。<br/>
いずれかの方法で作成をお願いします。<br/>

#### PCA給与シリーズから社員のアカウントを作成する

PCA給与シリーズから、社員のアカウントの作成、ライセンス登録を行うことができます。<br/>
こちらの方法は PCA Hub との連動設定の一部であるため、PCA Hub 給与明細のシステム管理者のみが行うことができます。<br/>

1. 「社員一覧」メニューを実行します。<br/>
![4](../assets/tenant_admin1.image4.png)<br/>

2. 社員一覧画面にて「ライセンス登録」メニューを実行します。<br/>
![5](../assets/tenant_admin1.image5.png)<br/>

3. ユーザー登録とライセンス付与を同時に行うか選択して実行します。<br/>
「ユーザー登録のみ行う」の実行後にライセンス付与を行う場合は「ユーザー登録とライセンス付与を行う」を選択してください。<br/>
![6](../assets/tenant_admin1.image6.png)<br/>

4. アカウントが登録され、社員と PCA Hub アカウントの紐づけが完了したことを確認してください。<br/>
![7](../assets/tenant_admin1.image7.png)<br/>

#### ポイント<!-- omit in toc -->

- チェックがONになっている社員を対象に、ライセンス登録を行います。
- PCA給与シリーズの社員マスターにメールアドレスが設定されている必要があります。
- 「ユーザー登録とライセンス付与を行う」を選択して実行した場合、給与担当者による[社員と PCA Hub のアカウントを紐づける](../system_admin/system_admin1.md#社員と-pca-hub-のアカウントを紐づける)操作は不要です。<br/>

#### 手動でアカウントを作成する

1. テナント管理サイトのユーザー一覧にアクセスします。
2. [新規] ボタンをクリックして新規入力ダイアログを表示します。
3. 社員情報の必須項目を入力します。
4. [保存] ボタンをクリックして入力内容を保存します。

![1](../assets/tenant_admin1.image1.png)

#### CSV ファイルからアカウントを作成する
テナント管理者と給与明細システム管理者が異なる場合の作成方法です。<br/>

1. [給与明細配信のための設定を行う](../system_admin/system_admin1.md) の[インポート用の CSV ファイルを作成する](../system_admin/system_admin1.md#インポート用の-csv-ファイルを作成する)にてCSVファイルを作成します。
2. テナント管理サイトのユーザー一覧にアクセスします。<br/>
3. アクションメニューから「ユーザーのインポート」をクリックして CSV ファイルを選択します。<br/>
![2](../assets/tenant_admin1.image2.png)<br/>

4. インポートが完了して結果ログがダウンロードされるまでしばらく待ちます。
5. ダウンロードされたログファイルを開いてインポート結果を確認します。<br/>
![3](../assets/tenant_admin1.image3.png)<br/>

### 社員自身がアカウントを作成する

テナント管理者がユーザー登録URLを発行し、社員に自身のアカウントを登録してもらうことができます。<br/>

1. テナント管理サイトのユーザー登録URLにアクセスします。<br/>
操作方法は[URLを利用したユーザーの追加](https://pca.jp/area_support/manual/hub/02_flow/flow_4.html)をご確認ください。<br/>
対象社員に自身のアカウントを登録していただきます。<br/>

2. 社員が登録したPCA Hub アカウントから、メールアドレスを受け入れます。<br/>
[こちら](../system_admin/system_admin1.md#社員自身にpca-hub-アカウントを登録してもらいメールアドレスを受け入れる)をご確認ください。<br/>

3. 「ライセンス登録」メニューを実行します。<br/>
![5](../assets/tenant_admin1.image5.png)<br/>

4. 対象社員を選択し、「ユーザー登録とライセンス付与を行う」を実行します。<br/>
![6](../assets/tenant_admin1.image6.png)<br/>

対象のPCA Hub アカウントに、アカウントの初期設定を行うためのメールと、ライセンス付与のメールが送信されます。<br/>

### PCA Hub アカウントが作成されたら

アカウントの初期設定を行うための手順は [PCA Hub にログインするための手順](../../startup.md#pca-hub-にログインするための手順) をご覧ください。

## 社員のアカウントを削除する

### PCA給与シリーズから社員のアカウントを削除する

PCA給与シリーズから、社員のアカウントの削除、ライセンス解除を行うことができます。<br/>
こちらの方法は PCA Hub との連動設定の一部であるため、PCA Hub 給与明細のシステム管理者のみが行うことができます。<br/>

1. 「社員一覧」メニューを実行します。<br/>
![4](../assets/tenant_admin1.image4.png)<br/>

2. 社員一覧画面にて「ライセンス解除」メニューを実行します。<br/>
![8](../assets/tenant_admin1.image8.png)<br/>

3. ライセンスを解除するか、ユーザーを削除するかを選択して実行します。<br/>
![9](../assets/tenant_admin1.image9.png)<br/>

### ポイント<!-- omit in toc -->

- チェックがONになっている社員を対象に、ライセンス解除を行います。

### 手動でアカウントを削除する

1. テナント管理サイトのユーザー一覧にアクセスします。<br/>
2. 削除したいアカウントにチェックを入れます。<br/>
![10](../assets/tenant_admin1.image10.png)<br/>
3. アクションメニューから[ユーザー情報の削除]を選択します。<br/>
![11](../assets/tenant_admin1.image11.png)<br/>
4. [削除] ボタンをクリックして削除を実行します。<br/>

### ポイント<!-- omit in toc -->

- ユーザーを削除した場合は元に戻すことはできません。同じ内容でユーザーを再登録しても異なるユーザーとして登録されます。<br/>
- ユーザーを使用する可能性がある場合は、ユーザーを削除せず [社員のライセンスを解除する](tenant_admin3.md#社員のライセンスを解除する) の操作を行います。<br/>
