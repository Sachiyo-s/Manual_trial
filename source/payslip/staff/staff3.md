# 配信された住民税特別徴収税額通知書を確認する<!-- omit in toc -->

ご自身宛に配信された住民税特別徴収税額通知書を閲覧またはダウンロードすることができます。

## 制限

- PCA Hub 給与明細のライセンスを付与されている必要があります。
- 住民税特別徴収税額通知書が公開されると社員に公開メールが配信されます。<br/>
サーバーの状況により、指定された日時よりも遅れてメールが配信される場合があります。<br/>

## 配信された住民税特別徴収税額通知書を閲覧する

1. PCA Hub 給与明細のクライアントサイトにアクセスします。<br/>
確認が必要な住民税特別徴収税額通知書がある場合、画面上部にバナーが表示されます。<br/>
![1](../assets/staff3.image1.png)<br/>

2. バナーをクリックすると照会ダイアログが表示されます。<br/>
「eLTAXでパスワードを取得する」をクリックして、eLTAXのサイトから照会に必要なパスワードを取得します。<br/>
サイトの使い方は、[eLTAXマニュアルコーナー＞ 特徴税通システムマニュアル ＞ 操作マニュアル ＞ 「eLTAX 特徴税通パスワード確認サイト操作マニュアル」](https://www.eltax.lta.go.jp/support/manual/)  をご確認ください。<br/>
![2](../assets/staff3.image2.png)<br/>

3. 照会ダイアログに戻り、パスワードを貼付して[照会]ボタンをクリックします。<br/>
![3](../assets/staff3.image3.png)<br/>

4. 展開が完了すると、明細一覧に住民税特別徴収税額通知書が追加されます。<br/>
![4](../assets/staff3.image4.png)<br/>

### ポイント<!-- omit in toc -->

- 配信された住民税特別徴収税額通知書は、公開されたことをメールにてお知らせします。<br/>

- サーバーの状況により、住民税特別徴収税額通知書の公開日時よりも遅れてメールが配信される場合があります。<br/>
クライアントサイトでは公開日時を過ぎれば公開されるため、メールが届く前であっても閲覧可能です。<br/>
