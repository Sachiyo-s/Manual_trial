# 32. ユーザーCSVフォーマット

## 全般

- データ形式：CSV（カンマ区切り）
- ファイル拡張子：.csv
- エクスポート時のファイル名：`ユーザー一覧_{yy}-{mm}-{dd}_{hh}-{mm}-{ss}.csv`
  - 例：ユーザー一覧_2024-06-13_09-30-58.csv
- 文字エンコーディング：UTF-8 (BOM付き)
- 改行文字：CRLF または CR または LF
  - エクスポートでは CRLF とする
- 行フォーマット
  - １行目：バージョン情報（省略可）
    - 形式は `"Ver" + {Major} + . + {Minor}` とする
      - Major や Minor は整数、Major は 1 以上、Minor は 0 ～ 9 を使う
      - 例：Ver1.0
    - 省略された場合は、最新バージョンと解釈する
      - 現在（2024/06）の最新バージョンは Ver1.0
  - ２行目：ヘッダー行（必須）
    - 項目一覧はバージョンごとに固定とする
  - ３行目以降：データ行
    - カンマ（,）を含む文字列データはダブルクォーテーション（"）で囲う

## データ項目

- [PCAアカウント](./PCAアカウント.md)

### Ver1.0

| No. | 項目名 | 必須 | 上限 | 制限 | 備考 |
| --: | -- | :--: | --: | -- | -- |
| 1 | アカウントID | | - | - | 常に空（予備項目） |
| 2 | ログイン名 | ◯ | 128文字 | 半角英数字 または 下記の記号のみ<br/>・ハイフン（`-`）<br/>・ピリオド（`.`）<br/>・アンダーバー（`_`）<br/>・アットマーク（`@`） | |
| 3 | メールアドレス | ◯ | 128文字 | RFCに準拠 <br/> 半角英数字 または 下記の文字のみ <br/> ・ハイフン（`-`）<br/>・ピリオド（`.`）<br/>・アンダーバー（`_`）<br/>・プラス（`+`）<br/>・パーセント（`%`） | RFC非準拠パターン<br/><abc..def@chatwork.com> ←連続ドット<br/><.abcdef@chatwork.com> ←先頭ドット<br/><abcdef.@chatwork.com> ←＠直前ドット |
| 4 | 表示名 | ◯ | 160文字 | 下記の記号を禁止<br/>・コロン（:）<br/>・ダブルクォーテーション（"） | |
| 5 | 姓 | ◯ | 20文字 |  下記の記号を禁止<br/>・コロン（:）<br/>・ダブルクォーテーション（"） | |
| 6 | 名 | | 20文字 |  下記の記号を禁止<br/>・コロン（:）<br/>・ダブルクォーテーション（"） | |
| 7 | 姓カナ | ◯ | 20文字 |  下記の記号を禁止<br/>・コロン（:）<br/>・ダブルクォーテーション（"） | |
| 8 | 名カナ | | 20文字 |  下記の記号を禁止<br/>・コロン（:）<br/>・ダブルクォーテーション（"） | |
