# PCA ID 組織を統合する

PCA Hub で利用する組織と、PCAクラウド／PCAサブスクで利用する組織は、１つの組織に統合することができます。

- [PCA Hub をご利用中](./PCA%20Hub%20から利用を始める.md)のお客様が、PCAクラウド／PCAサブスクから [PCA ID 組織を作成](./PCA%20ID%20組織を作成する.md)して PCA ID を利用している場合
- [PCAクラウド／PCAサブスクで PCA ID をご利用中](./PCAクラウド／PCAサブスクから利用を始める.md)のお客様が、追加で PCA Hub を利用する場合

上記の状況では、組織を統合することで、どちらのサービスからも共通の組織を利用できるようになります。  
結果として、より大きな粒度で、組織内のユーザーや利用サービスに対する一元的な管理が可能となります。

組織の統合手順は、統合元となる組織から、統合先に対する、統合リクエストの送信によって開始となります。  
統合リクエストを受信した統合先の組織が、そのリクエストを承諾することにより、組織の統合が実施されます。  
どちらの手順も、それぞれの組織における組織管理者が実行します。

詳しくは、「[組織を統合する（統合元組織）](../組織管理者向け/組織を管理する/組織を統合する（統合元組織）.md)」、および「[組織を統合する（統合先組織）](../組織管理者向け/組織を管理する/組織を統合する（統合先組織）.md)」をご確認ください。
