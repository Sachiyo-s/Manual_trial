PCA Hub から利用を始める

PCA Hub における PCA ID とは？

PCA Hub から利用を始める場合、PCA ID を特別に意識する必要はありません。
PCA Hub には PCA ID が統合されているため、PCA Hub では常に PCA ID によってユーザーを認証します。
詳しくは「ログインの流れ」メニューから、利用可能なログイン方法をご確認ください。

  • パスキーを使用した簡単なログイン方法

  • パスワードと認証コードを使用した２段階ログイン方法

  • パスワードとバックアップコードを使用した２段階ログイン方法

PCA Hub からユーザーを作成すると、PCA ID において PCA アカウントが作成され、PCA Hub ユーザーと紐づきます。
パスキーやパスワード、バックアップコード等の認証情報は、PCA ID が管理し、PCA アカウントが保持します。
複数の PCA Hub テナントを利用する場合であっても、１つの PCA アカウントを使い、一度のログイン操作でどのテナントも利用することができます。

PCA Hub から PCA ID 機能を利用する

一般ユーザーが PCA ID 機能を利用する

セキュリティを設定する

ユーザー情報画面の［基本情報］タブから、ログイン方法の［設定］ボタンをクリックすると、ご自身の認証情報を設定できます。
従来方式のパスワードやバックアップコードの他に、最新の方式としてパスキーを利用することができます。

パスワードについて詳しくは、「パスワードの作り方」、「パスワードを変更する」をご確認ください。
バックアップコードについて詳しくは、「バックアップコードを発行する」をご確認ください。
従来方式のパスワードを使ったログイン方法は、「パスワードと認証コードを使用した２段階ログイン方法」、または「パスワードとバックアップコードを使用した２段階ログイン方法」をご確認ください。

最新方式となるパスキーについて詳しくは、「パスキーとは」、「パスキーを登録する」をご確認ください。
パスキーを使ったログイン方法は、「パスキーを使用した簡単なログイン方法」をご確認ください。

 

サービス利用状況を確認する

ユーザー情報画面の［ログ情報］タブから［ログイン履歴を表示］ボタンをクリックすると、ご自身の利用状況を確認できます。
詳しくは「自身のサービス利用状況を確認する」をご確認ください。

テナント管理者が PCA ID 機能を利用する

ログイン状況を確認する

テナント管理画面の［ダッシュボード］メニューにおいて、［ログイン状況を確認］ボタンをクリックすると、PCA ID 組織内ユーザーの利用状況を確認できます。
詳しくは「ユーザーのサービス利用状況を確認する」をご確認ください。

必要となる権限

ユーザーのサービス利用状況を確認するには「組織管理者」の権限が必要です。
詳しくは「権限と役割」をご確認ください。

PCA Hub テナント作成時は、初回の PCA Hub テナント管理者が、自動的に PCA ID 組織管理者となります。
2人目以降の PCA ID 組織管理者は、明示的に権限を付与する必要があります。

ユーザー作成時の注意事項

複数の PCA Hub テナントから同じメールアドレスのユーザーを作成することで、１つの PCA アカウントを利用することができます。
このとき、２人目以降のユーザー作成で入力した情報は、PCA アカウントとの紐づけにのみ利用し、１人目のユーザー情報を優先して維持します。

２人目以降のユーザー作成で利用される入力項目は次の通りです。

  • 利用される入力項目

    • メールアドレス

    • ログイン名

  • 利用されない入力項目

    • ユーザー名

    • 姓名・カナ

初回のユーザー作成においては、アカウント設定リクエストをメール送信し、パスワードの設定やバックアップコードの発行が行われます。
２人目以降のユーザー作成においては、メールアドレス確認リクエストをメール送信し、入力したメールアドレスの本人確認のみが行われます。

ユーザー作成時の上書き

後からユーザーを作成するときの入力情報によって、登録済みのユーザー情報を上書きする場合があります。
最初に作成したユーザーが、どのサービスも利用していない場合が、これに該当します。
このときは、後から作成するユーザーの入力情報が優先となりますが、パスワード等のアカウント設定は不要となります。
