給与明細を受信する準備を行う

PCA Hub 給与明細で明細を受信するための準備を行います。

制限

  • PCA Hub のアカウントが作成されており、PCA Hub に招待されている必要があります。

  • PCA Hub 給与明細のライセンスを付与されている必要があります。

PCA Hub アカウントの初期設定を行う

はじめて PCA Hub アカウントを登録するときは

PCA Hub アカウントの初期設定を行います。


【PCAサービス】≪会社名≫ からのアカウント設定リクエストというタイトルのメールを開いてください。

≪会社名≫ には、PCA Hub 給与明細を契約している会社名が入ります。

PCA Hub にログインするための手順 ＞ [PCA ID]をご利用の場合の操作 ＞1. 招待メールの受信 を確認し、手続きを行ってください。

【PCAサービス】{会社名} からのアカウント設定リクエスト というメールが見つからないときは

  • 【PCA Hub サービス】≪会社名≫からのパスワード設定リクエストが届いている場合

    PCA Hub にログインするための手順 ＞ [PCA ID]をご利用でない場合の操作 ＞ 1. 招待メールの受信 を確認し、手続きを行ってください。

  • 【PCAサービス】メールアドレス確認リクエスト が届いている場合

    一般ユーザー向け ＞メールでメールアドレス確認リクエストが届いたら を確認し、手続きを行ってください。

ポイント

  • 対応しようとしたら、期限が切れていた場合

    招待から一定時間経過していると、招待期限が切れてしまいます。
    管理者へ再送信を依頼してください。

  • 設定したパスワードを忘れてしまった場合

    ログイン画面に、「パスワードを忘れた場合はこちら」や「パスワードをお忘れの方」というリンクがあります。 こちらから、ご自身でパスワードの再設定ができます。
    参考）パスワードの作り方

  • 【PCA Hub 給与明細】≪会社名≫ からのライセンス付与のお知らせ というメールは何か

    PCA Hub 給与明細を利用するためのライセンス付与のお知らせです。
    この後の　同意内容を確認する で使用します。

同意内容を確認する

給与担当者から PCA Hub 給与明細で明細を配信することに対する同意案内が配信されています。
同意事項に同意しないと PCA Hub 給与明細をご利用いただけません。

(1)  PCA Hub 給与明細のクライアントサイトにアクセスします。

  • PCA Hub からのライセンス付与メールに記載されているURLからアクセスする場合、URLをクリックします。

  • ポータルサイトからアクセスする場合、[PCA Hub 給与明細] ボタンをクリックします。

(2)  同意案内および同意事項の全文を閲覧した上で「同意事項を確認しました。」のチェックをONにします。
「同意事項」はスクロールして全文を表示する必要があります。

(3)  ［同意する］ ボタンをクリックします。

ポイント

  • 【PCA Hub 給与明細】≪会社名≫ からのライセンス付与のお知らせ というタイトルのメールが見つからない、誤って削除したかもしれない場合

    テナント名がわかる場合は、URL を参考に給与明細クライアントサイトにアクセスすることができます。
    テナント名がわからない場合は、管理者にお問い合わせください。
