# 組織名を変更する

:::info 権限による制限

組織情報の変更は、組織管理者のみ行うことができます。

:::

:::caution `PCA Hub` をご利用中の組織

PCA Hub を利用中の組織は、組織名を変更できません。  
PCA ID の組織名は、必ず PCA Hub テナント名と一致します。

:::

## (1) 管理コンソールへのアクセス

PCA ID 管理コンソールにアクセスするため、ブラウザーで、<https://id.pca.jp/orgs> にアクセスします。  
管理コンソールの画面を表示するには、組織管理者の権限が必要となります。

## (2) 組織の選択

現在の組織を確認し、必要があれば組織を選択します。

組織管理者の権限をもつ複数の組織に所属している場合、処理を実行したい組織を選択します。  
アクセス直後は、前回の組織が選択されています。

![組織の選択](../images/組織の選択.png)

## (3) カテゴリの選択

［組織管理］カテゴリを選択します。

![組織管理カテゴリ](./images/組織管理カテゴリ.png)

## (4) 組織名の変更開始

組織名の右側にある［変更］ボタンをクリックします。

![組織名の変更開始](./images/組織名の変更開始.png)

## (5) 組織名の変更

組織名を入力して、［変更］ボタンをクリックします。  
組織名は、30文字以内で、英数小文字、およびハイフン（-）のみ使用できます。

![組織名の変更](./images/組織名の変更.png)

:::info 制限事項

他の組織と同じ組織名、または以前に使用されたことのある組織名を使うことはできません。  
また、PCAクラウド／PCAサブスクから作成した組織の場合、先頭の「org-」を含めない組織名にはできません。  
[PCA Hub の組織へ統合する](../../ご利用までの流れ/PCA%20ID%20組織を統合する.md)ことで、PCA Hub テナント名と同じ組織名にすることができます。

:::
