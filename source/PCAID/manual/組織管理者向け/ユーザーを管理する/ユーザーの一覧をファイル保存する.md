# ユーザーの一覧をファイル保存する

:::info 権限による制限

ユーザー一覧のファイル保存は、組織管理者のみ行うことができます。

:::

## (1) 管理コンソールへのアクセス

PCA ID 管理コンソールにアクセスするため、ブラウザーで、<https://id.pca.jp/orgs> にアクセスします。  
管理コンソールの画面を表示するには、組織管理者の権限が必要となります。

## (2) 組織の選択

現在の組織を確認し、必要があれば組織を選択します。

組織管理者の権限をもつ複数の組織に所属している場合、処理を実行したい組織を選択します。  
アクセス直後は、前回の組織が選択されています。

![組織の選択](../images/組織の選択.png)

## (3) カテゴリの選択

［ユーザー管理］カテゴリを選択します。

![ユーザー管理カテゴリ](./images/ユーザー管理カテゴリ.png)

## (4) ユーザーの選択

ユーザー一覧から対象にするユーザーを探して、左端のチェックをオンにします。

ユーザー件数が多い場合、検索機能を利用してユーザーを絞り込んでから探します。  
ユーザー一覧ページ内の選択ユーザーに対して、最大で100人まで、まとめて対象にすることができます。  
未選択であれば、対象ユーザーを選択するように案内を表示します。

![ユーザー一覧からの選択（チェックボックス）](./images/ユーザー一覧からの選択（チェックボックス）.png)

:::tip ユーザーの検索

検索対象となるユーザー情報は次のとおりです。大文字・小文字は区別しません。

- ユーザー名
- 姓・名
- 姓・名カナ
- ログイン名
- メールアドレス

:::

:::tip ユーザーの一括選択と解除

見出しの左端にあるチェックをオン／オフにすることで、ページ内のユーザー選択をまとめてオン／オフすることができます。

:::

## (5) ユーザーエクスポートの開始

［アクション］メニューから、［ユーザーのエクスポート］をクリックします。

![ユーザーエクスポート](./images/ユーザーエクスポート.png)

## (6) ユーザーエクスポートの実行

確認画面で［OK］ボタンをクリックし、ユーザー CSV ファイルをダウンロードします。

![ユーザーエクスポート確認](./images/ユーザーエクスポート確認.png)

ユーザー CSV ファイルは、［ダウンロード］フォルダーを保存先として、ファイル名「`ユーザー一覧_{年}-{月}-{日}_{時}-{分}-{秒}.csv`」で保存します。

![ユーザーエクスポートファイル](./images/ユーザーエクスポートファイル.png)

ユーザー件数に応じてダウンロードに時間がかかります。  
ユーザー CSV ファイルの内容について、詳しくは「[PCA ID ユーザー CSV ファイル仕様](../../ご利用の前に/PCA%20ID%20ユーザーCSVファイル仕様.md)」をご確認ください。
